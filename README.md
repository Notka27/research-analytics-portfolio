## Проект 1: Credit Risk Analysis — прогнозирование дефолта заемщиков (Kaggle)

**Цель:** построить и сравнить модели машинного обучения для прогнозирования дефолта (AUC-ROC 0.93), выявить ключевые факторы риска.

**Данные:** 32 581 запись, 12 признаков (возраст, доход, кредитная история, цель кредита и др.).

**Что сделано:**
- Проведён полный цикл анализа: EDA, очистка, кодирование, масштабирование.
- Построены и сравнены 3 модели: логистическая регрессия (0.846), Random Forest (0.933), Gradient Boosting (0.929).
- Выявлены ключевые факторы дефолта: кредитный класс, доход, цель кредита, история дефолтов.

**Инструменты:** Python (pandas, numpy, scikit-learn, matplotlib, seaborn).

**Результат:** Random Forest — точность предсказания дефолта 97%, AUC-ROC 0.933.

## Проект 2: Bondora Risk Analysis — реальные P2P-кредитные данные

**Цель:** построить честную модель предсказания дефолта на реальных данных без утечек (data leakage).

**Данные:** 134 529 записей, 112 признаков, пропуски до 90%.

**Что сделано:**
- Отбор 35 релевантных признаков, очистка и заполнение пропусков (группировка по стране, медиана, мода).
- Преобразование категориальных скорингов в числовые шкалы (Equifax, MicroL, AsiakasTieto).
- Выявлена и устранена утечка данных (признаки активной просрочки).
- Построена честная модель Random Forest (AUC-ROC 0.809).

**Результат:** модель без утечек данных с AUC-ROC 0.809, точность 75%, precision дефолта 68%.

## Проект 3: План исследования влияния зарплатного проекта на доход банка 

**Цель:** разработать методологию оценки влияния перехода состоятельных клиентов на зарплатный проект на доход банка.

**Что сделано:**
- Определены критерии «состоятельный клиент», структура доходов и затрат банка.
- Спроектирован сбор данных по 6 блокам (транзакции, остатки, продукты, доходы, демография).
- Предложены методы: Propensity Score Matching, факторный анализ, кластеризация, байесовские сети.
- Разработана стратегия на оперативный (1–3 мес.) и стратегический (1–2 года) уровни.
- Включены расчёты ROI, Payback Period, NPV и этапы внедрения.

**Результат:** комплексный план исследования, готовый к внедрению в банковской среде.
